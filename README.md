# Maintenance (TYMIO)

Статическая страница «сайт на техническом обслуживании» для экосистемы **TYMIO** (децентрализованный протокол структурированных продуктов). Показывается посетителям, когда основной сайт временно недоступен.

## Назначение

- Одностраничное приложение на React с сообщением: *The site is undergoing maintenance.*
- Брендинг Tymio (логотип) и ссылки на соцсети/каналы (иконки Bebra, Telegram, X).

## Стек

- **React** 18
- **Create React App** (`react-scripts` 5)
- Сборка без source maps (`GENERATE_SOURCEMAP=false`)

## Скрипты npm

| Команда | Описание |
|--------|----------|
| `npm start` | Режим разработки, порт **5110** |
| `npm run dev` | Режим разработки, порт **7001** |
| `npm run preprod` | Локальный просмотр в стиле preprod, порт **5410** |
| `npm run build` | Production-сборка в каталог `build/` |

## Сборка и раздача статики

Скрипт `build.sh` (Unix):

1. Выполняет `npm run build`
2. Копирует результат в `build-serve/` (каталог `build/` после копирования удаляется)

Для локальной раздачи собранной статики можно использовать `serve`:

- `servestart.sh` — порт **5410** (как у `preprod`)
- `servestart.dev.sh` — порт **7001** (как у `dev`)

Требуется глобально установленный [`serve`](https://www.npmjs.com/package/serve) или аналог.

## Структура

- `src/` — компонент `App`, стили, SVG-иконки
- `public/` — `index.html`, манифест, мета-теги Open Graph для TYMIO

## Примечание

В `package.json` имя пакета указано как `maintance` (опечатка); каталог репозитория — `maintenance`.
